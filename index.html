Configuration des APIs

🏛️ API France Travail (obligatoire pour les offres)

L'API France Travail (ex Pôle Emploi) est gratuite. Pour obtenir vos identifiants :
1. Rendez-vous sur francetravail.io
2. Créez un compte et inscrivez votre application
3. Copiez votre Client ID et Client Secret

✨ API OpenAI (optionnel — pour l'analyse IA)

L'analyse IA utilise gpt-4o-mini (économique). Pour obtenir une clé :
1. Créez un compte sur platform.openai.com
2. Allez dans API Keys et créez une nouvelle clé
3. Copiez la clé (commence par sk-)

🔒 Vos clés sont stockées uniquement dans votre navigateur (localStorage). Elles ne sont jamais envoyées ailleurs.
